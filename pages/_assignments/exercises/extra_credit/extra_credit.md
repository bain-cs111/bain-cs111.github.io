>**NOTE**: This assignment is completely optional and not required.

This is an optional extra credit review assignment that can add up to 30 points to your exercise grade in this class (which currently has 800 points from the 8 highest scoring exercises).

The intent is to help you study for Quiz 4 by writing some more complex functions that span different topics in the course.

This assignment is due at 9:00am on December 8th. No exception. No extensions.

* * *

## Getting Started

There are six functions in the below template `rkt` file for you design. You'll need to use the type signature and purpose statements in order to complete the six functions. Each is worth the same amount of credit.

If you decide not to attempt a particular function, just leave its body as the string `"Fill me in!"`. Remember, if your program has syntax errors, the autograder won't be able to run it.

All the details you need to complete the assignment are in the RKT file. Make sure to not rely ONLY upon the built-in tests. Consider what other tests might be useful to make sure your functions do what you say they do.

## Turning It In

Before turning your assignment in, **run the file one last time to make sure that it runs properly and doesn’t generate any exceptions**, and all the tests pass (unless you chose to not implement some particular function). Make sure you've also spent some time writing your OWN `check-expect` calls to test your code.

Then, make sure to read the [Autograder Guide](https://canvas.northwestern.edu/courses/178849/pages/whats-an-autograder) one last time.
